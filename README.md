## Usage

Simply open `index.html` in any modern web browser. No installation or server required.

### Navigation

1. **Search**: Type in the search box to find specific antibodies, blood groups, or systems
2. **Filter by System**: Use the dropdown to filter by specific blood group systems
3. **Filter by Clinical Significance**: Filter antibodies by HDFN risk, HTR risk, or hemolytic potential
4. **Click System Cards**: Click on any system card to expand and view antibodies
5. **Click Antibodies**: Click on individual antibodies to view detailed information
